## 1. Gate Precedenti

- [ ] G1 (Wireframe/Scope) approvato
- [ ] G2 (Mockup) approvato
- [ ] G3 (UAT) approvato — Signoff ricevuto

---

## 2. Bug e Issue

- [ ] Nessun bug Blocker aperto
- [ ] Nessun bug Major aperto
- [ ] Bug Minor accettati per post go-live (documentati)
- [ ] Issue GitLab tutte in Done o documentate come rinviate

---

## 3. Contenuti

- [ ] Contenuti definitivi inseriti
- [ ] Immagini ottimizzate
- [ ] Testi revisionati
- [ ] SEO base (title, description, alt)
- [ ] Privacy policy / Cookie policy presenti

---

## 4. Infrastruttura Produzione

- [ ] Ambiente produzione pronto
- [ ] Dominio configurato
- [ ] Certificato SSL attivo
- [ ] DNS preparato (TTL abbassato se necessario)
- [ ] Backup configurato
- [ ] Monitoraggio attivo

---

## 5. Migrazione (se applicabile)

- [ ] Freeze contenuti attivo
- [ ] Import iniziale completato
- [ ] Delta/aggiornamenti finali pianificati
- [ ] Redirect configurati e testati
- [ ] Verifica URL critici

---

## 6. Documentazione Pronta

- [ ] Go-live plan preparato
- [ ] Rollback plan preparato e testato
- [ ] Smoke test checklist pronta
- [ ] Contatti emergenza aggiornati

---

## 7. Comunicazioni

- [ ] Cliente informato di data/ora go-live
- [ ] Stakeholder avvisati
- [ ] Comunicazione post go-live preparata
- [ ] Eventuale pagina manutenzione pronta

---

## 8. Team

- [ ] Team disponibile il giorno del go-live
- [ ] Reperibilità post go-live confermata
- [ ] Ruoli e responsabilità chiari

---

## 9. Approvazioni

- [ ] Approvazione go-live dal cliente (scritta)
- [ ] Approvazione interna (se richiesta)
